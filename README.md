# tokia-loh

TOKIA-LOH est une application e-commerce développée en React.js, permettant aux utilisateurs de parcourir des catégories de produits, ajouter des articles à un panier, passer commande avec paiement à la livraison et laisser une note de commande. L’interface adopte un design moderne aux couleurs bleu ciel et violet, avec une navigation fluide.
